# HR Bot (Lite, SQLite-first)

Лёгкий Telegram‑бот на **NestJS + Prisma (SQLite) + Telegraf**:

- **Q&A база знаний**: добавление, список, удаление и **поиск по ключевым словам**.
- **Excel‑уведомления**: загрузка `.xlsx` из Telegram (со 2‑й строки), создание событий и рассылка уведомлений в заданные дату/время.
- **Безопасность**: HTTP‑эндпоинты защищены `API_KEY` (через `Authorization: Bearer ...`). Доступ к командам бота ограничен списком `ALLOWED_CHAT_IDS` в `.env`.
- **Оптимизация под SQLite**:
  - Парсер Excel работает транзакционно и массово (`createMany`) — быстро и без лишних round‑trip.
  - Поиск реализован с ранжированием в памяти (нормализация + баллы) — корректно и регистронезависимо для любых языков.
  - Индекс на `(delivered, at)` ускоряет выборку ожидающих уведомлений.
- **Минимализм**: без Docker, без лишних зависимостей — только то, что нужно для продуманного UX.

Команды бота: `/start`, `/question`, `/searchquestion`, `/add`, `/delete`, `/upload`, `/myid`.
